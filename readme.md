# 云学堂
### 云学堂是一款基于微服务架构的在线教育网站

# 前后端分层架构说明

## 一、展现层（前端）

**技术栈：**  
- Vue  
- WebPack  
- Axios  
- Pinia  
- Vue-Router  
- Element-UI  
- Echarts  
- Sass  

**安全防护：**  
- XSS 防护  
- CSRF 防护  

---

## 二、接入层

**主要功能：**  
- 反向代理  
- CDN  
- 身份认证  
- 请求路由  
- 负载均衡  

---

## 三、服务层

### 微服务组件：
- Nacos  
- Seata  
- OpenFeign  
- Sentinel  
- LoadBalancer  
- SpringCloud  

### 分布式组件：
- XXL-JOB  
- Redisson  
- RabbitMQ  

---

## 四、数据层

**数据库类型：**  
- **关系型数据库：** MySQL  
- **NoSQL 数据库：** Redis、MongoDB  
- **搜索引擎：** Elasticsearch  

---

## 五、DevOps（开发运维）

- 文档管理：YAPI  
- 代码管理：GOGS  
- 持续集成：Jenkins  
- 容器管理：Docker  
- 日志管理：GrayLog  
- 链路追踪：SW  

---

## 六、第三方服务

- 阿里云  
- 腾讯云
